# OMC pruebaBack

This project is the backend side which provides the frontend project with endpoints as an REST API with connection to a remote Database which is MongoDB Atlas Database.

## Installation

1. Do `npm i` command on the console on the project main route.
2. Is ready for launch!

## Usage

1. Launch the project with `npm run dev` if you want to run it with Nodemon or `npm run start` if you want to launch it as normal mode.
2. You can also try the online version!
